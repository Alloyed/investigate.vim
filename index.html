Investigate is a plugin for viewing documentation.

You can use investigate to view the documentation for the current word in Vim in a few different places. Through your web browser, using Dash.app on OS X, or with an arbitrary shell command.

Installation

With Vundle

In your .vimrc: 
Bundle 'Keithbsmiley/investigate.vim'
Then run :BundleInstall from within Vim

With Pathogen

cd ~/.vim/bundle
git clone \
  https://github.com/Keithbsmiley/investigate.vim.git

Usage

Once investigate is installed you can setup a mapping for it in your .vimrc. I would recommend something like: 
nnoremap K :call investigate#Investigate()
